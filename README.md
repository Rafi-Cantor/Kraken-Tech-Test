# Kraken Tech Test

## Getting started

First clone the repo and activate the virtual environment.

Install the requirements using:
````
pip install -r requirements.txt
````
Add a `.env` file to the root of the project and add this to it:

````
API_KEY=your-api-key
````

## Run the program
````
python main.py
````

## Run the tests
````
pytest test.py
````
